# stamp-art

Slack のカスタム絵文字（スタンプ）を使ってモザイクアートを生成する CLI ツール。

入力画像をグリッド状に分割し、各セルに最も色の近いスタンプを当てはめることで、スタンプだけで構成されたアート画像を作れる。Slack に貼り付けられる `:emoji:` 形式のテキストファイルも出力可能。

## セットアップ

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

`.env` に Slack Bot Token を設定する（`emoji:read` スコープが必要）。

```
SLACK_TOKEN=xoxb-your-token-here
```

## 使い方

### 1. スタンプ画像を取得

```bash
python -m src fetch-stamps
```

Slack ワークスペースのカスタム絵文字を一括ダウンロードして `data/stamps/` に保存する。

### 2. スタンプの色情報を計算

```bash
python -m src process-stamps
```

各スタンプを 2x2 の 4 象限に分割し、それぞれの平均色と不透明率を計算してキャッシュする。この処理は一度だけ実行すればよい。

### 3. モザイクアートを生成

```bash
python -m src generate -i input.jpg -g 50 --slack
```

| オプション       | 説明                              | デフォルト                    |
| ---------------- | --------------------------------- | ----------------------------- |
| `--input`, `-i`  | 入力画像パス                      | (必須)                        |
| `--grid`, `-g`   | グリッドサイズ（長辺のマス数）    | 30                            |
| `--output`, `-o` | 出力ファイルのベース名            | `output/{timestamp}_mosaic_{grid}x{grid}` |
| `--cell-size`    | 出力画像の 1 セルのサイズ (px)    | 32                            |
| `--no-duplicate` | 隣接セルで同じスタンプを使わない  | on                           |
| `--slack`        | Slack 貼り付け用の `.txt` も出力  | on                           |
| `--min-opacity`  | スタンプの最低不透明率 (0.0〜1.0) | 0.5                           |
| `--no-animated`  | アニメーションスタンプを除外      | off                           |

## マッチングアルゴリズム

単純な 1 色の平均色ではなく、スタンプと入力画像の各セルの両方を **2x2 の 4 象限に分割**し、各象限の平均色（計 12 次元）のユークリッド距離で比較する。これにより空間的な色分布を考慮した精度の高いマッチングができる。

## フィルタ

### 透過フィルタ (`--min-opacity`)

Slack のテーマ（ライト / ダーク）によって透過スタンプの見え方が変わるため、不透明率の低いスタンプを除外できる。デフォルトは 50% 以上。

### アニメーション除外 (`--no-animated`)

GIF アニメーションなど動きのあるスタンプを除外する。モザイクの見た目を落ち着かせたいときに使う。
